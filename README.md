# @e18e/deopt

A collection of tools for tracing, visualizing and debugging deoptimization in Node.js/v8.

## Install

```sh
npm i -g @e18e/deopt

# or just npx it
npx @e18e/deopt foo.js
```

## Usage

```sh
npx @e18e/deopt [options] <script.js> [-- <args>]
```
